# SytemMonitor
O System Monitor é uma solução leve e eficiente para monitoramento em tempo real de infraestrutura de TI, focada em Servidores Físicos (Hosts) e Máquinas Virtuais (VMs). Desenvolvido para facilitar o dia a dia de administradores de rede, o painel oferece uma visão clara do status de disponibilidade (UP/DOWN) de ativos críticos.
